## Study Science Built In

This tool is designed around evidence-based learning techniques:

- **Active Recall** — Flashcards and practice problems force retrieval, which strengthens memory 2–3x more than re-reading
- **Spaced Repetition** — Study plan rotates topics across sessions rather than marathoning one subject
- **Interleaving** — Mixed practice problems across topics improve long-term retention
- **The Testing Effect** — Attempting problems before reading solutions produces better exam performance
